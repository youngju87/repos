### Glossary of Terms

- **Session:** A group of user interactions within a 30-minute window
- **Bounce Rate:** Percentage of sessions with only one page view
- **Conversion Rate:** Percentage of sessions that result in a goal completion
- **AOV (Average Order Value):** Average revenue per transaction
- **ROAS (Return on Ad Spend):** Revenue generated per dollar spent on advertising
- **MoM (Month-over-Month):** Comparison to previous month
- **YoY (Year-over-Year):** Comparison to same month last year

## WORD/POWERPOINT FORMATTING GUIDE

### For Word Document:

1. **Page Setup:**
   - Margins: 0.75" all sides
   - Landscape orientation for wide tables
   - Portrait for text-heavy sections

2. **Typography:**
   - Headers: Bold, brand colors
   - Body: 11pt, clear font
   - Tables: 10pt for readability

3. **Colors:**
   - Primary: #0f172a (headers)
   - Accent: #2563eb (data highlights)
   - Red for negative trends, Green for positive trends

4. **Charts/Screenshots:**
   - Leave placeholder boxes for charts
   - Export from GA4/Looker Studio and paste
   - Ensure all charts have clear labels and legends

5. **Tables:**
   - Header row with brand color background
   - Alternating row colors
   - Right-align numbers
   - Use %, $, and other formatting consistently

### For PowerPoint:

1. **Slide Master:**
   - Create consistent template with brand colors
   - Header with client name and reporting period
   - Footer with page numbers

2. **One Topic per Slide:**
   - Executive Summary: 1-2 slides
   - Each major section: 1-2 slides
   - Focus on visual data representation

3. **Charts:**
   - Use native PowerPoint charts or paste from GA4
   - Keep simple and clear
   - One main insight per chart

4. **Speaker Notes:**
   - Add detailed notes to each slide
   - Include context and analysis
   - Preparation for presentation
